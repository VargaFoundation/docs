Vorath is a CSI plugin that lets Kubernetes workloads mount HDFS volumes through the standard CSI interface. It aims for a simple installation experience, sensible defaults, and smooth production operations.

## What you'll find here

- Version compatibility (Kubernetes, CSI, Hadoop)
- Prerequisites and sizing
- Installation with Helm and example manifests
- Configuration, security, and best practices
- Day-2 operations, monitoring, and troubleshooting

Goal: make HDFS easily accessible to cloud-native platforms, without compromising on performance or security.
